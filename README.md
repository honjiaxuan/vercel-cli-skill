# OpenCode Skill: Vercel CLI

A comprehensive OpenCode skill for managing and configuring Vercel Projects directly from the Command-Line Interface (CLI).

## Overview

This skill empowers AI agents and developers to interact with Vercel's platform programmatically. It covers the full range of Vercel CLI capabilities, from basic deployments to advanced infrastructure management like DNS, SSL certificates, and team coordination.

## Features

- **Deployment Management**: Deploy to production or preview environments with ease.
- **Environment Syncing**: Synchronize local environment variables with Vercel secrets.
- **Live Documentation**: Fetch official Vercel docs as markdown directly from the terminal.
- **Project Linking**: Link local directories to existing Vercel projects.
- **Domain & DNS Control**: Manage domains, buy new ones, and configure DNS records.
- **Security**: Manage SSL certificates and team access scopes.
- **CI/CD Friendly**: Detailed guidance on using Vercel CLI in automated pipelines.

## Installation

To use this skill in your OpenCode environment, copy the `SKILL.md` file to your local skills directory:

```bash
mkdir -p ~/.config/opencode/skills/vercel-cli
cp SKILL.md ~/.config/opencode/skills/vercel-cli/
```

## Usage

Once installed, the skill is automatically discovered by OpenCode agents. You can trigger it by asking the agent to:

- "Deploy this project to Vercel production"
- "Link this directory to my Vercel project"
- "Sync my environment variables with Vercel"
- "Add a new domain to my Vercel project"
- "Fetch the Vercel documentation for 'edge-functions'"
